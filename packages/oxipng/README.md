## Installation

```bash
npm install @wasm-codecs/oxipng
```

**Requirements:**
- Node.js 10 or later

## Usage

```typescript
import encode from '@wasm-codecs/oxipng';

(async () => {
  const encodedImage = await encode(image, inputInfo, encodeOptions);
})();
```
